# CityGuessGameProject

Greetings and welcome to my second GitHub project: the City Guessing Game!
In this game, you attempt to guess the country, province, and city of a mystery city selected from the "world-cities.txt" file. 
You're given the letters of the country, province, and city in sequential order after every guess. Please note that this game is still a WIP and works best on Google Chrome! 
In order to play this game, you must do the following: 
- You must have the WolframAlpha and the Maps Javascript APIs. For the WolframAlpha API, input your API key on line 5 of server.js and line 185 of src/CityGuessingGame.html. For the Google Maps API, input your API key inside the quotes on line 1030 of src/CityGuessingGame.html, line 524 of bin/CityGuessingGame.html, line 226 of bin/index.html, and line 226 of bin/index.html
- install node and npm and then run "node -v" and "npm -v"
- run the command "npm install express cors". This is crucial to get it to work on your browser.
- run the command "node server.js". This will run a server on localhost:3000.
- Open a separate terminal and run "npm install -g live-server" and then after that run "live-server" This should bring you to http://127.0.0.1:8080/.
- To get to the game, click src and then "CityGuessingGame.html". Enjoy the game!
